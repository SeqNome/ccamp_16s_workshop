# 16S Metagenomics Workshop Materials

This repository contains the core materials used in the **16S Metagenomics Workshop**.

The contents are organized to help participants:
- follow the hands-on sessions,
- explore the datasets,
- and practice basic analysis workflows.

---

## Folder Descriptions

### slides.html

This is the **final workshop presentation**.

- Open this file in any modern web browser (Chrome / Firefox / Edge).
- Contains all conceptual material, figures, and explanations.

---

### data/

This folder contains **example datasets** used during the workshop.

These files are used to:
- demonstrate basic data exploration,
- perform normalization and visualization,
- and practice simple analysis workflows.

Participants are encouraged to:
- inspect these files,
- load them into R or other tools,
- and experiment with them during the sessions.

---

### scripts/

This folder contains **analysis scripts** used in the workshop.

These scripts demonstrate:
- basic command-line operations,
- simple R-based data handling,
- exploratory data analysis (EDA),
- and visualization.



---

## How to Use This Repository

1. Download the repository (ZIP or clone).
2. Open `slides.html` in your browser.
3. Follow along with the presentation.
4. Use files in `data/` with scripts in `scripts/`.
